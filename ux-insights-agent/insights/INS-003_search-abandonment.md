# INS-003: Mobile Search Results Cause High Abandonment

## Statement
Users searching on mobile abandon results pages at a high rate because the layout makes it difficult to evaluate and compare options.

## Evidence

### Analytics (Q1 2025)
Mobile search result pages show a 58% exit rate, versus 29% on desktop. Users who exit from search results do not return to search in the same session 71% of the time.

### Usability Sessions (Round 3, March 2025)
In sessions where participants were asked to find a specific product via search, 6 of 10 mobile users expressed frustration with the results layout. Common issues:
- Product images were too small to evaluate at a glance
- Price and key attributes were not visible without tapping into each result
- No way to filter results without scrolling to the bottom of the page

### Heatmap Data (Feb 2025)
Heatmaps on mobile search results show almost no interaction below the fold. Users interact with the top 3 results only, then exit.

## Argument
Mobile users cannot efficiently evaluate search results because the layout does not surface enough information at a glance. Unlike desktop, where users scan and compare across a grid, mobile forces sequential tapping — a pattern that increases cognitive load and time-to-decision. The high exit rate and low return-to-search behavior suggests users are not finding what they need and giving up rather than refining their search. The same sessions that surfaced checkout issues (Round 3) also surfaced this problem, suggesting mobile UX quality is a systemic issue across the purchase funnel.
