# INS-004: Users Don't Like the Dashboard

## Statement
Users find the dashboard confusing and hard to use.

## Evidence

### Usability Sessions
Some users in recent sessions said the dashboard was overwhelming. A few people mentioned they weren't sure where to look first. One person said it was confusing.

### Survey
We ran a survey and some users gave low scores for the dashboard section.

## Argument
Because users are confused, the dashboard probably needs to be redesigned. The feedback was pretty consistent that people don't understand it. This is likely affecting how much they use the product overall.
